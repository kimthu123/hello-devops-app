# Hello DevOps App

A simple bash script showcasing DevOps fundamentals:  
- Print username, date/time, and working directory  
- Demonstrates version control with Git (branches, commits, merge)  
- Deployed workflow ready for next steps into CI/CD & cloud infra  

## Tech Stack & Tools
- Bash shell script (`main.sh`)  
- Git & GitHub (version control and branching)  
- WSL Ubuntu on Windows 11 (local development environment)  
- To be extended: Docker, CI/CD pipelines, AWS or other cloud  

## How to Run Locally
```bash
cd ~/DevOps-Learning/week1_linux_git
chmod +x main.sh
./main.sh
